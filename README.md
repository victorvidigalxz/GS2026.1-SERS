# GS2026.1-SERS
Sistema inteligente de monitoramento energético para uma missão espacial experimental.

# Objetivo

O projeto tem como objetivo monitorar dados simulados de uma missão espacial, analisando:

- Temperatura
- Energia solar
- Bateria
- Comunicação
- Consumo energético

O sistema gera alertas automáticos e realiza decisões básicas para aumentar a eficiência energética e a sustentabilidade da operação espacial.

# Funcionalidades

- Monitoramento de dados simulados
- Alertas automaticos
- Modo economia de energia
- Verificação da comunicação
- Controle de temperatura
- Análise de eficiência energética
